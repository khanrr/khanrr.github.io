Privacy on the Internet

Rian Khan

Introduction

Many people browse the internet daily to read the news, watch and stream videos, or talk to friends. Most of the time, these are done using online platforms like Facebook, Youtube, and others. What most people who browse the web tend to forget is privacy and how vulnerable personal privacy is on the web. With companies that offer the aforementioned platforms harvesting user data and information, personal privacy is always compromised. Fortunately with the help of FOSS (free open-source software) and various contributors around the world, securing your own privacy on the web is easy. There are several recommendations that have been made by others regarding securing privacy. One popular resource for privacy recommendations is the Privacy Tools website. This website contains recommendations regarding web browsers, search engines, and other tools we use to browse the web.

Browser Recommendations

Several recommendations regarding which web browser should be used to browse the internet have been made. The top recommendation for a private web browser is Mozilla Firefox. Using Firefox with the various settings described on the Privacy Tools website and in different hardening guides will ensure a private web browsing experience. This means a complete removal of ads and trackers which are abundant on the internet. Another recommendation for a private web browser if a browser similar to Google Chrome is preferred is Brave browser. Brave was created by ex-Mozilla CEO Brendan Eich as a private alternative to Chrome. Compared to Firefox, Brave requires less tweaking to become private as it was designed to be private out of the box. With a built-in ad-blocker and other measures, Brave browser offers around the same level of privacy as Firefox, but using the Chromium engine as opposed to the Gecko engine that Firefox runs on. However, it does not offer the same level of customization that is offered by Firefox due to the limitations of the Chromium engine. Choosing between the two depends on how much effort you are willing to put in towards tweaking your browser's settings and creating a private browsing experience.

Additionally, it is also recommended to test your browser of choice using some of the tools listed on the aforementioned website. One recommended tool is Browser Leaks which tests your browser and shows what data can be discovered. Another good tool to use is Cover Your Tracks by EFF.org which tests your browser and shows how you can be tracked while using your browser. The results from these tools will offer a deeper look at how your browser protects your privacy.

Other Recommendations

If you wish to secure your privacy further, the Privacy Tools website also offers recommendations regarding browser extensions, search engines, email clients, VPNs and other tools.

If you wish to keep your current browser of choice, installing some of the recommended extensions will help improve privacy. One top recommendation for all browsers is UBlock Origin, which is an ad-blocker and tracker blocker in one extension. The extension can be customized to block ads and trackers that have been engineered to bypass the default settings. Another good extension to use is Cookie AutoDelete. This extension deletes cookies that are not being used upon closing a tab. The extension can be customized further to keep specific cookies and delete others. Additionally, if your browser of choice does not offer an option to automatically upgrade HTTP to HTTPS, then the extension HTTPS Everywhere is one that is highly recommended to be installed. However, as many browsers are starting to include the function natively, this extension is slowly becoming redundant. As a result, EFF (Electronic Frontier Foundation) is slowly shutting down the extension.

If the only change you want to make is your choice of search engine, there are a few recommendations that have been made in that area as well. The top recommendation for search engines is Startpage, a search engine based in the Netherlands. Startpage delivers the same results as Google, but with extra measures in place to preserve privacy. Another good recommendation is DuckDuckGo, a search engine based in the US. DuckDuckGo uses its own engine to return results. Using either of these options will make your browsing more private as opposed to using a search engine like Google.

For email services, recommendations include ProtonMail (a mail service hosted in Switzerland), Tutanota (a mail service hosted in Germany), and StartMail (a mail service hosted in the Netherlands). All of these services are great alternatives to popular mail services like Gmail and Outlook.

If using a VPN is desired, some recommended VPN services include Mullvad (VPN service from Sweden), ProtonVPN (VPN service from Switzerland), and IVPN (VPN service from Gibraltar). These VPN services utilize a variety of encryption methods to make your browsing more private and secure. However, it should be acknowledged that using a VPN while browsing will not make your browsing anonymous.

For more information and recommendations, go to Privacy Tools.
